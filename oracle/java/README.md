# Java Oracle

This directory contains the Java oracle used for parity checks against feasible
small generated cases.

Tracked source and support files:

- `src/`: Java source.
- `lib/acm.jar`: required library for the oracle build.

Generated output:

- `out/`: Java class files generated by setup or manual compilation. This path
  is ignored by git.

Run the maintained parity wrapper from the repository root:

```bash
tests/java_parity.sh
```
